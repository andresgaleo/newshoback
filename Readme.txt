Se realiza el proyecto de prueba técnica NewShore.

Arquitectura usada hexagonal. 

Se uso una función recursiva que pudiese consultar todas las rutas disponibles de los vuelos 

En la aplicación front end se debe configurar la ruta en el app.component.ts
Se uso angular con material ui.

INSTALACION

Para la instalación simplemente se requiere clonar la aplicación y ejecutarla y desde el front end configurar la ruta.

Para las pruebas unitarias simplemente se debe correr y debe funcionar sin problemas. Se trabajo con Xunit.
